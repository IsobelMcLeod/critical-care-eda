# Exploratory Analysis of Intensive Care Hospital Patients

This project explores clinical data from critically ill patients admitted to five US medical centres.

## Project Overview
The analysis focuses on:
- Assessing data quality
- Cleaning and preparing the dataset
- Performing exploratory data analysis (EDA)
- Identifying patterns and outliers relevant to patient care and outcomes

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Notes
This project was developed in a Jupyter Notebook as part of academic coursework and portfolio development.
